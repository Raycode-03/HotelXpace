# Hotel Booking App

A hotel room booking system built with Next.js and Supabase and mapbox showing the  hotels and the locations side by side.

## Features

- QR code room access
- Session-based guest flow
- Admin and staff roles
- Menu browsing with filters
- Background email jobs
- Pagination and responsive UI

## Tech Stack

- Next.js
- Nest.js
- Mapbox
- Supabase
- React Query
- Radix UI
- Sonner

## Running Locally

1. Clone the repository

git clone <repo>

2. Install dependencies

npm install

3. Add environment variables

Create `.env.local`
check the rest of the variables from `.env.example`
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

4. Run the development server

npm run dev

App runs at:
http://localhost:3000
